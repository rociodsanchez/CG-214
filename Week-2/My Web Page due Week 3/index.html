Who am I?

I'm a sixth semester junior at the Fashion Institute of Technology. I study Advertising and Marketing Communications.
I grew up in Queens and went to a performing arts high school for instrumental music. I like fashion, and I love business, and I think I can really get ahead with knowledge of coding as well as best business practices.

Why is coding important for me?

I find it really fun to code! I just don't know how to build anything from scratch, but I learned from last class that that's not necessary and that a lot of programmers actually work with things that are already built. In addition, when studying programming on my own, I found that the real work in this field is debugging, not building from scratch. So I find it important to learn proper syntax and how to properly find errors in code. On a less personal note, I work for an ecommerce company called Beth Macri that uses Liquid, a Ruby-based programming language, but it also has JavaScript assets as well as CSS and basic HTML. It's really fun to go into the code at work and know exactly what I'm changing and why it didn't work in the first place. Having solid programming skills can really put a person ahead in the job market.
