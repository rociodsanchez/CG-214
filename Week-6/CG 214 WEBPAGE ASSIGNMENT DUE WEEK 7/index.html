Week 5

Class notes from September 28

  • The Cascading Style Sheet! CSS!
    • Make sure every page in your site is always linked to the main .css file (unless you want another sheet to be linked to a specific page? Depends!)
  • Comments in CSS look like /* this */!
  • To selection a tag within another tag (Child Selector), use tag>tag!
  • See all different kinds of selectors below:
Selector Type Tag Example Meaning
Universal Selectors * { } Applies to full document
Type Selectors h1, h2, h3 { } Targets individual elements
Class Selector .note { } Targets all classes with value note
ID Selector #name { } Targets only one ID with value name
Child Selector li>a { } Targets all anchor tags within lists
Descendent Selector p a { } Targets all anchor tags in paragraphs, even if there's other elements between them

Week 1

Class notes from August 31

  • Two main languages
    • HTML - HyperText Markup Language
    • CSS - Cascading Styling Sheets
  • Let's learn about HTML Elements
    • HTML Elements are composed of
      • A start tag, like <h1>
      • An attribute, like the id="head" in <h1 id="head">
      • The content, like the "Hi Rocio!" in <h1 id="head">Hi, Rocio!</h1>
      • The closing tag, like the </h1> in <h1 id="head">Hi, Rocio!</h1>
    • Tags
      • Structural tags are
        • <!doctype html>
        • <html></html>
        • <head></head>
        • <body></body>
        • <div></div>
      • Text tags are
        • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
        • <p>
        • <b>
        • <i>
        • <ul>
        • <ol>
        • <li>
        • </ br>
      • The link tag
        • <a>
        • The a is attributed to a URL (Uniform Resource Locator), relative URL's in the same site, or linking to a specific part of the page using #
        • You may add attribute target="blank" if you want the page to open in a new window
      • The image tag
        • <img>
        • Attributed src to where the image is located, it can be at a URL, a local directory, or inside your own site
        • To change size, add weight and height attributes
        • The alt attribute is used for SEO and accessibility, and it describes the image
      • Directories are important!
        • The root folder for this entire class website is called CG 214 WEBPAGE
